# PowerPuran YouTube Bot

**Official Homepage of PowerPuran YouTube Bot**

PowerPuran YouTube Bot is an automation tool that helps creators upload and manage YouTube videos with AI assistance.

## Important Links
- **Privacy Policy:** https://powerpuran1-droid.github.io/powerpuran-bot/privacy

## About the App
This application accesses your YouTube account only with your permission to:
- Upload videos  
- Generate titles and descriptions  
- Create thumbnails  
- Automate publishing workflow  

You can revoke access anytime from your Google Account security settings.
